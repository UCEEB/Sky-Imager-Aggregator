Detailed Description

GSM modbus library.

LibrforPiV2.

Library includes the functions needed the RaspberryPi for work GSM modem and modbus irradiance sensor

Version
1.0
Author
Jan Havrlant

Script at a given interval access the camera, get image and storage to local storage. In given interval send thumbnail and log file to remote storage via GSM modem
